Frequently Asked Questions

How does SemCraft generate papers? +

SemCraft uses Python and Pandas in the backend to randomly select questions from a Master Question Bank Excel file according to the specified schema, ensuring a well-balanced paper every time.

Can I review the paper before downloading? +

Yes! Once you generate a paper, you are taken to a "Preview" page where you can read all the questions. If you don't like them, you can simply generate a new one.

Are the downloaded PDFs formatted correctly? +

Absolutely. The PDF generation ensures that the downloaded file contains the exact same questions as your preview, strictly formatted with college headers and proper academic spacing.

Who built SemCraft? +

SemCraft is a student-built project aimed at simplifying academic preparation while showcasing clean frontend design and full-stack integration.
